# Notes Sharing Portal

A full-stack web application where users can upload, share, and access notes.
The project includes a responsive UI and REST APIs built with Node.js and Express.js.

## Status
Frontend, backend, and API code will be uploaded soon.

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express.js
- MongoDB
